# SpeedContrOtter Extension
YouTubeやPrime Videoなどの動画再生速度をコントロールするChrome拡張機能です．

## 特徴
- キーボードショートカットで再生速度の変更
- 画面上に再生速度を表示
- YouTubeやPrime Videoをはじめとする多くの動画サイトに対応
- 動画を変更しても再生速度を維持
- **ポップアップ画面でカワウソのアニメーションを表示**

## インストール方法
2種類のインストール方法(**ZIPファイルのダウンロード**，**リポジトリのクローン**)があります．

### 方法 1: ZIPファイルのダウンロード
1. GitHubリポジトリページで **"Code"** -> **"Download ZIP"** の順にクリック．
1. ZIPファイルをPCの好きなところに解凍する．
1. 拡張機能管理ページを開く．
    1. 検索窓に`chrome://extensions/`と入力する．
    1. またはブラウザ画面右上のメニューボタンから **"拡張機能"** -> **"拡張機能を管理"**
1. 画面右上の**デベロッパーモード**をオンにする．
1. 画面左上の**パッケージ化されていない拡張機能を読み込む**をクリックし，解凍したフォルダを選択する．

### 方法 2: このリポジトリをクローンする
1. 以下のコマンドでローカルにリポジトリをクローンする
    ``` bash
    git clone https://github.com/nukko0529/speed_controller.git
    ```
1. 拡張機能管理ページを開く．
    1. 検索窓に`chrome://extensions/`と入力する．
    1. またはブラウザ画面右上のメニューボタンから **"拡張機能"** -> **"拡張機能を管理"**
1. 画面右上の**デベロッパーモード**をオンにする．
1. 画面左上の**パッケージ化されていない拡張機能を読み込む**をクリックし，解凍したフォルダを選択する．

## 使用方法
### キーボードショートカット
動画の再生中，以下のショートカットが有効になります．
- **d** -> 再生速度を上げる
- **s** -> 再生速度を下げる
- **r** -> **1倍速**に戻す

再生速度の変更幅はポップアップ画面から調整できます．
また，動画の再生中は常に右上に再生速度が表示されます．

---

### ポップアップ画面
拡張機能アイコンをクリックしてポップアップをひらきます．
- **速度の変更幅の調整** (e.g., 0.05 / 0.1 / 0.25 / etc.)
    - **Current step**はそのときの変更幅
    - スライダーで好みの変更幅を決めたら**Apply**ボタンで適用
- **カワウソのアニメーション再生** 再生速度に応じたアニメーションが再生されます．
    (寝る -> 歩く -> 走る -> 泳ぐ)
